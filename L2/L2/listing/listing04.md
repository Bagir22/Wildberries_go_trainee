Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

    
	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа выведт числа от 0 до 9, и дальше будет пытаться читать из не закрытого канала, из-за чего будет deadlock

```
